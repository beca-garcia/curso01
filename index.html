O que você irá aprender no curso

A seguir, estão descritos os tópicos que serão abordados durante o curso.

O que é Agilidade

Esta seção irá falar sobre a definição do que é ser ágil, trazendo exemplos práticos do uso da agilidade para execução de atividades e construção de produtos e serviços.

Manifesto Ágil

Declaração de princípios que fundamentam o desenvolvimento ágil de software.

Framework Scrum

O Scrum é um framework que ajuda pessoas, equipes e organizações a gerarem valor através de soluções adaptativas para problemas complexos.

Scrum Eventos

Cada evento no Scrum é uma oportunidade para inspecionar e adaptar o que está sendo feito durante a Sprint.

Scrum Papéis

Todos os profissionais que fazem parte do Scrum Team, possuem papéis bem definidos, onde cada integrante trará sua contribuição dentro da Sprint. Os times são estruturados e capacitados para gerir o seu próprio trabalho entregando valor a cada entrega para o cliente ou produto.

Scrum Artefatos

Toda documentação e incrementos produzidos quando usamos o framework Scrum.

Métricas

Dentre as métricas que um Scrum Master deve dominar para poder analisar os dados e acompanhar o progresso das entregas, temos: O gráfico de Burndown e o Gráfico de Velocity.

Atividades Práticas

Todos participarão de uma simulação de uma Sprint, onde cada um terá um papel dentro do Time Scrum e será praticado todo o conteúdo teórico para fixação. Além de atividades de análise de graficos de Burndown e Velocity.
